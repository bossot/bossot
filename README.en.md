## 🏗️ My homelab

Homelab managed with a **GitOps + IaC** philosophy, documented using the [Diátaxis](https://diataxis.fr/) framework on MkDocs.

### ☸️ Platform

- **k3s** cluster with 6 nodes (3 control plane + 3 workers) managed by **FluxCD**.
- **Ingress** with Traefik and TLS termination.
- **Secrets** managed externally.

### 🌐 Networking & security

- Split DNS with redundant Pi-hole instances (public vs internal domains).
- VLAN segmentation: LAN, SRV, DOCKER, DNS, K8S.
- Defense in depth across 3 layers: firewall, cloud proxy access controls, reverse proxy security policies.

### 💾 Backup

- **3-2-1** strategy with local, NAS, and cloud copies.
- Local snapshots and weekly external-drive air-gapped backup.
