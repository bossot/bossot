## 🏗️ El meu homelab

Homelab gestionat amb filosofia **GitOps + IaC**, documentat en format [Diátaxis](https://diataxis.fr/) sobre MkDocs.

### ☸️ Plataforma

- Cluster **k3s** de 6 nodes (3 control plane + 3 workers) amb **FluxCD**.
- **Ingress** amb Traefik i TLS.
- **Secrets** amb gestió externa.

### 🌐 Xarxa i seguretat

- DNS dividit amb Pi-hole redundant (dominis públics vs interns).
- Segmentació per VLANs: LAN, SRV, DOCKER, DNS, K8S.
- Defensa en profunditat en 3 capes: firewall, proxy al núvol amb controls d'accés, reverse proxy amb polítiques de seguretat.

### 💾 Backup

- Estratègia **3-2-1** amb còpia local, NAS i núvol.
- Snapshots locals i còpia setmanal en disc extern (air-gap).
